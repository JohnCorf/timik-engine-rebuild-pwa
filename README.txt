TIMIK Agriculture Engine Rebuild Records - Mobile PWA V1

Access code:
lurch2026

Built from the Workshop Job Card Generator PWA base and specialised for TIMIK Agriculture engine rebuild documentation.

Main features:
- Engine rebuild record/job card
- Customer database
- Saved engine jobs
- Parts used/required table
- Workshop photos, up to 3 per job
- Rebuild stage sign-off checklist
- Daily workshop diary linked to saved engine jobs
- Weekly report built from diary entries
- Print / Save PDF support
- Email generation for engine records and weekly reports
- Workshop settings with logo, phone, email and footer text
- Dark/navy industrial workshop styling
- Mobile-first layout for iPhone/iPad use
- Offline PWA support
- Footer/branding: Powered by SouthWorx

How to use locally:
1. Extract this ZIP.
2. Open index.html in a browser.
3. Enter access code: lurch2026.
4. Fill in an engine rebuild record.
5. Save the engine job before adding diary entries.
6. Add daily diary entries against saved jobs.
7. Open Weekly Report, choose a week and build/export the report.

Create a new GitHub repository:
1. Go to GitHub.
2. Create a new public repository, for example: timik-engine-rebuild-pwa.
3. Do not add a README, .gitignore or licence during creation.
4. Upload these files into the new repository root:
   - index.html
   - styles.css
   - app.js
   - manifest.json
   - service-worker.js
   - icon.svg
   - README.txt
5. Commit to main.
6. Go to Settings > Pages.
7. Source: Deploy from a branch.
8. Branch: main / root.
9. Save and wait for GitHub Pages to publish.

Important:
This is intentionally lightweight. It does not include stock control, invoicing, accounts, payroll or cloud syncing.
